Login

Pada halaman ini pengguna diharapkan masuk menggunakan NIP dan password yang telah di tentukan di awal dengan menggunakan NIP pengguna untuk passwordnya.

Edit Profil Admin

Pada halaman dashboard admin dapat melihat jumlah user yang telah terdaftar, jumlah file yang telah diupload, jumlah skp yang telah dibuat, jumlah nilai perilaku yang telah dinilai, dan tanggal periode SKP yang telah ditentukan sebelumnya.

Edit Profil Admin

Pada halaman ini admin dapat mengubah detail profil dengan cara memasukkan atau mengubah isi pada textfield Form kemudian klik submit dan muncul notifikasi diatas “Update data successfully!”.

Daftar SKP dan File Pegawai

Pada halaman ini admin dapat melihat daftar SKP dari setiap pegawai yang telah terdaftar di sistem. Tidak hanya itu, admin juga dapat melihat file yang disimpan oleh pegawai di tombol aksi ikon file.

Daftar SKP berdasarkan tahun

Pada halaman ini setelah menekan tombol SKP pada daftar pegawai di halaman SKP & File Pegawai akan di tunjukkan menuju pilihan tahun dan periode Januari – Juni atau Juli – Desember.

SKP Pegawai

Pada halaman ini admin dapat melihat isi SKP dari pegawai yang dilihat dan bisa mencetak SKP tersebut.

Data File

Pada halaman ini setelah menekan tombol aksi file akan ditujukan menuju halaman data file dari setiap pegawai, para pegawai bisa menyimpan file file penting seperti akta kelahiran, KTP, CPNS, dan lain-lain.

Data Pegawai

Pada halaman ini menampilkan daftar pegawai dengan role yang sudah ditentukan, menambahkan pewagai, mengubah role pegawai menjadi pejabat penilai, admin dapat mengubah detail pegawai.

Data Admin

Pada halaman ini admin dapat melihat daftar data admin unit kerja yang terdaftar di sistem. Sama seperti di halaman pegawai.

Data Unit Kerja

Pada halaman ini admin dapat melihat daftar unit kerja yang terdaftar oleh admin.

Jadwal

Pada halaman ini admin mengatur jadwal pengisisan target SKP dan pengisian realisasi SKP. Sebelum mengatur jadwal, terlebih dahulu melihat apakah periode yang dipilih benar periode Januari – Juni atau Juli – Desember.

Dashboard Unit Kerja

Pada halaman dashboard admin unit kerja dapat melihat jumlah user yang telah terdaftar menurut unit kerja, jumlah file yang telah diupload, jumlah skp yang telah dibuat, jumlah nilai perilaku yang telah dinilai, dan tanggal periode SKP yang telah ditentukan oleh admin.

Edit Profil Unit Kerja

Pada halaman ini admin unit kerja dapat mengubah detail profil dengan cara memasukkan atau mengubah isi pada textfield Form kemudian klik submit dan muncul notifikasi diatas “Update data successfully!”.

SKP dan File Pegawai

Pada halaman ini admin unit kerja dapat melihat daftar SKP dari setiap pegawai menurut unit kerja admin unit kerja. Tidak hanya itu, admin unit kerja juga dapat melihat file yang disimpan oleh pegawai di tombol aksi ikon file.

Data Pegawai

Pada halaman ini menampilkan daftar pegawai dengan role yang sudah ditentukan, menambahkan pegawai, mengubah role pegawai menjadi pejabat penilai, admin dapat mengubah detail pegawai.

Dashboard Pejabat Penilai

Pada halaman ini pejabat penilai hanya dapat melihat profil diri sendiri dan menu hanya ada Dashboard, SKP, dan File.

Edit Profil Pejabat Penilai

Pada halaman ini pejabat penilai dapat mengubah detail profil dengan cara memasukkan atau mengubah isi pada textfield Form kemudian klik submit dan muncul notifikasi diatas “Update data successfully!”.

Form SKP

Pada halaman ini pejabat penilai hanya dapat melihat SKP yang sudah diisi atau belum diisi target atau realisasinya.

Pengukuran SKP

Pada halaman ini muncul ketika jadwal yang ditentukan telah berlaku. Terdapat menu Pengukuran, Nilai Perilaku dan Nilai Perilaku Kerja yang membedakan dari halaman pada saat jadwal target SKP.

Nilai Perilaku

Pada halaman ini pengguna dapat melihat nilai perilaku mereka, apakah sudah diisi oleh pejabat penilai masing masing atau belum.

Nilai Perilaku

Pada halaman ini pertama tama menampilkan daftar pegawai mana yang dinilai oleh pejabat penilai ini atau pengguna.

Pengukuran SKP

Pada halaman ini muncul ketika jadwal yang ditentukan telah berlaku. Terdapat menu Pengukuran, Nilai Perilaku dan Nilai Perilaku Kerja yang membedakan dari halaman pada saat jadwal target SKP.

Nilai Perilaku

Pada halaman ini pengguna dapat melihat nilai perilaku mereka, apakah sudah diisi oleh pejabat penilai masing masing atau belum.

Data Pegawai Penilaian

Pada halaman ini pertama tama menampilkan daftar pegawai mana yang dinilai oleh pejabat penilai ini atau pengguna.

Dashboard Pegawai

Pada halaman ini pegawai hanya dapat melihat profil diri sendiri dan menu hanya ada Dashboard, SKP, dan File.

Edit Profil Pegawai

Pada halaman ini pegawai dapat mengubah detail profil dengan cara memasukkan atau mengubah isi pada textfield Form kemudian klik submit dan muncul notifikasi diatas “Update data successfully!”.

SKP Pegawai

Pada halaman ini pegawai hanya dapat melihat SKP yang sudah diisi atau belum diisi target atau realisasinya.

File Pegawai

Pada halaman ini pegawai dapat mengisi file yang bisa disimpan oleh mereka sendiri apabila admin membutuhkan file pegawai ini, langsung bisa melihat file tersebut pada akun admin atau admin unit kerja.

Form SKP

Pada halaman ini pegawai memilih pejabat penilainya siapa, kemudian dibawahnya mengisi tugas jabatan yang akan dikerjakan selama 1 periode, pegawai juga bisa mengubah apabila ada kesalahan penulisan atau menghapusnya. Kemudian yang terakhir mengisikan nilai target SKP dengan jumlah tugas jabatan yang sudah diisi.

Nilai Perilaku

Pada halaman ini pegawai dapat melihat nilai perilaku mereka, apakah sudah diisi oleh pejabat penilai masing masing atau belum.
